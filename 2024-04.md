    - F 有个复杂度分析，剩下是牛魔题。

      ```
      for i (1, n + 1):
      	for j (i, n + 1, i):
      		for k (j, n + 1, k):
      			//O(1)
      ```

      这个东西是 $2\log$ 的，因为你考虑他其实是： $$\sum_{i=1}^{n}\frac{n}{i}\log(\frac{n}{i})$$ ，把 $\log$ 提外面去，就是调和级数再多乘个 $\log$ 。
